# Secure File Sharing System with Encryption

## Overview
This project aims to build a secure file sharing platform where files are encrypted before being stored on the server. The system ensures safe file transfer between users using encryption techniques.

## Features
- User authentication
- Secure file upload
- File encryption before storage
- Secure file download
- Access control for shared files

## Technologies
- Python
- Flask
- AES Encryption
- MySQL
- HTML / CSS

## Future Improvements
- End-to-end encryption
- File integrity verification using hashing
- Temporary download links
- Activity logging
